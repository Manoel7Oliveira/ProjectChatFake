# Project Chat Fake

Aplicação front-end desenvolvida com HTML, CSS e JavaScript que simula uma interface de chat com envio de mensagens, respostas automáticas e interação entre usuários.

## 🚀 Tecnologias

- HTML5
- CSS3
- JavaScript (ES6+)

## 📌 Funcionalidades

- Envio de mensagens
- Respostas automáticas
- Pesquisa de mensagens
- Reações com emojis
- Notificação sonora
- Interface inspirada em aplicativos de chat

## 💡 Conceitos praticados

- Manipulação do DOM
- Delegação de eventos
- Criação dinâmica de elementos
- Event Listeners
- Manipulação de arrays e objetos
- Pesquisa e filtragem de dados
- Organização do código em funções

## 📚 Aprendizados

Este projeto foi desenvolvido com o objetivo de praticar manipulação do DOM, eventos, funções, arrays e lógica de programação utilizando JavaScript.
